# Accessing theme palette

Inside your component, you can retrieve the theme palette and variant.

```jsx
import { useTheme, Badge } from '@material-ui/core';

const App = () => {
  const theme = useTheme();
  console.log(theme.variant);
  return <Badge color="theme.palette.primary.main" />;
};
```

# Overriding palette defaults

With `@tecsinapse/ui-kit` it's possible to override default styles for components. The following example override the Picker toolbar background color.
Check [material-ui docs](https://material-ui.com/) for component CSS API rules.

```jsx
import { ThemeProvider } from '@tecsinapse/ui-kit';

const defaultStyle = {
  MuiPickersToolbar: {
    toolbar: {
      backgroundColor: '#FFF',
    },
  },
};

const App = () => (
  <ThemeProvider variant="orange" overrides={defaultStyle}>
    <App />
  </ThemeProvider>
);
```
